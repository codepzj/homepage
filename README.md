## homepage

这是一个由`nextjs`和`tailwindcss`构建的个人主页，使用了`shadcn/ui`组件库。

## 项目结构

```
src/
├── app/
│   ├── page.tsx
│   └── layout.tsx
├── components/
│   ├── theme-switcher.tsx
│   └── card.tsx
├── config/
│   └── site.config.ts
```

## 修改配置

在`src/config/site.config.ts`中修改配置，然后重新运行项目。

## 项目依赖

```
pnpm install
```

## 项目运行

```
pnpm run dev
``` 

## 项目部署

```
pnpm run build
```

## 项目部署

```
pnpm run deploy
```
